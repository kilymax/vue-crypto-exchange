## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/kilymax/vue-crypto-exchange
   ```
2. Install dependencies
   ```sh
   npm install
   ```
3. Run app in dev mode
   ```
   npm run dev
   ```
